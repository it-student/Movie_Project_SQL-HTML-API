## Launch

* Simply Download this project and create a virtual environment.
* Create your own API-Key on omdbapi.com and put ypur key as 'OMDB_API_KEY' into your own '.env'-file
* Next to your newly generated API-Key within your newly created '.env'-file, 
  set 'BASE_URL' as described on omdbapi.com
* install all dependencies with the help of requirements.txt
* run main.py
